# Roamingoo — Luxury Travel Landing (Static)

This is a static, responsive premium travel landing site built with pure HTML, CSS and vanilla JavaScript.

Quick start:

1. Place optimized assets in `assets/` (see note below).
2. Open `index.html` in your browser.

Recommended assets (place in `assets/`):
- `hero.mp4` — short (5–12s) optimized MP4 H.264 cinematic video (autoplay, muted). Keep < 5MB for fast load.
- `hero-poster.jpg` — poster image for the video.
- `destination1.jpg`..`destination4.jpg` — destination images (optimized JPG/WEBP).
- `cta.jpg` — background for CTA section.
- `reviewer1.jpg`..`reviewer3.jpg` — small circular reviewer photos.

If you don't have assets yet, the layout will still render. Replace the sample files with your own cinematic video and images to get the full premium experience.
